📖 回測執行順序（無前瞻偏誤）
Step1 → 以昨日持倉計算今日報酬（不含今日新進場）→ 更新淨值

Step3 → 依今日收盤判斷進場 → 扣買入費用 → 加入持倉

版本 ①

原版（基準）

出場：跌破5MA 且 OBV當日斜率由正轉負（AND）
OBV每1~2天就觸發，但需AND才出場

版本 ②　⬆ 最佳

最短持有 5 個交易日

在原版基礎上加一條規定：
進場後前 5 個交易日不看出場訊號
→ 給股票一週時間發展，減少短期震盪踢出

版本 ③　⬇ 較差

OBV 5日EMA出場

OBV做5日指數平滑後再看斜率是否由正轉負
理論上更穩定，但實際上：
→ EMA滯後讓我們賣太慢，損失比原版更大

🔍 為什麼策略沒賺到錢？

⚠ 問題1：OBV 訊號太敏感

  • 原版 OBV 定義：昨天OBV上漲、今天下跌 → 觸發多轉空
  • 這在日線資料幾乎每 1~2 天就發生一次
  • 策略1 在 485 個交易日共出場 904 次（≈每天 1.9 次）
  • 頻繁買賣 × 手續費 0.57% = 每次進出侵蝕本金

❌ 問題2：進出場均線不對稱

  • 進場條件：站上 20日均線（中期趨勢）
  • 出場條件：跌破 5日均線（超短期波動）
  • 買進眼光長，賣出耐心短 → 選到好股也沒時間漲
  • 最短持有5日版本改善了這個問題（S1: -7.0% → -0.1%）

🔬 OBV EMA5 為何反而更差？

  • EMA5 平滑了 OBV，但同時引入 2~3 天的滯後
  • 股價跌破5MA很快（2~3天），但EMA5確認轉空需要5~7天
  • 等待EMA5確認期間，股價繼續下跌 → 每筆損失更大
  • S2 均報酬：原版 -0.09% → EMA5版 -0.61%（更差）

✅ 最短持有5日 為何有效？

  • 進場後5日內完全不看出場訊號 → 直接阻擋短期噪音觸發
  • 出場次數：S1 904 次 → 863 次（↓41 次）
  • 平均持倉：S1 14.8檔 → 19.6檔（更充分分散）
  • S1 改善幅度最大：-7.0% → -0.1%（幾乎回到平盤）

🔬 條件6「近5日漲幅」門檻敏感度分析

什麼是這個分析？ 條件6「近5日漲幅 < X%」原本設定為 5%，懷疑在牛市中把強勢股擋掉了。 固定其他條件，從 3% 一路測到無限制，找出能讓策略贏大盤的設定值。

📌 關鍵發現

策略2（近10日）在門檻 ≥ 50% 時可超越大盤

  • 門檻「< 50%」：S2報酬 +105.5%，超越大盤 +5.9%
  • 門檻「無限制」：S2報酬 +108.2%，超越大盤 +8.6%

策略1（近3日）在此期間無法超越大盤

  • 即使完全移除條件6，S1 最高報酬 +90.2%，仍低於大盤 +106.1%
  • 近3日籌碼在 2024-2026 牛市選股效果較弱，建議搭配最短持有5日一起改
